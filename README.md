# AlpukatGH.githup.io
Dalam game ini, pemain berada di sebuah galeri seni digital yang gelap. Pemain harus menemukan pintu keluar dengan hanya menggunakan kursor sebagai senter. Tantangannya adalah beberapa "lukisan" di dinding bisa berubah menjadi monster jika dilihat terlalu lama, atau jika kursor menyentuhnya.
